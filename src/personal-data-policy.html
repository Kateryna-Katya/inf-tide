Политика обработки персональных данных

1. Общие положения

Настоящая политика обработки персональных данных определяет порядок и условия обработки персональных данных, предпринимаемые проектом Inf-Tide.blog (далее — «Оператор»), и устанавливает меры по обеспечению безопасности этих данных.

1.1. Важнейшей целью и условием осуществления своей деятельности Оператор считает соблюдение прав и свобод человека и гражданина при обработке его персональных данных.

2. Основные понятия

  • Веб-сайт — совокупность материалов, доступных в сети интернет по адресу Inf-Tide.blog.
  • Пользователь — любой посетитель веб-сайта.
  • Персональные данные — любая информация, относящаяся к Пользователю (Email, телефон, имя).

3. Данные, которые мы обрабатываем

Оператор может обрабатывать следующие категории данных:

  • Фамилия, имя;
  • Адрес электронной почты (Email);
  • Номера телефонов;
  • Обезличенные данные (cookie, IP-адрес) через сервисы статистики.

4. Цели обработки

  • Идентификация Пользователя.
  • Установление обратной связи (консультации, заявки).
  • Информирование Пользователя (с его согласия). Пользователь всегда может отказаться от рассылки, направив письмо на адрес support@inf-tide.blog.
  • Аналитика и улучшение качества сайта.

5. Безопасность и хранение

  • Оператор принимает все необходимые меры для защиты данных от неправомерного доступа.
  • Персональные данные никогда не будут переданы третьим лицам, за исключением случаев, предусмотренных законом.
  • Пользователь может актуализировать свои данные или отозвать согласие, написав на support@inf-tide.blog.

6. Заключительные положения

Пользователь может получить любые разъяснения по вопросам обработки данных, обратившись к Оператору по почте support@inf-tide.blog.

Этот сайт использует файлы cookie для улучшения пользовательского опыта. Ознакомиться с политикой.
